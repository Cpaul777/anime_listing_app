# Anime Listing App

## Background
  This project was for me to practice react and javascript along with utilizing APIs.

## Features
- Search function
- Trending List based on most searched term
- Mobile UI 

## How to run 
Run this command in your terminal 
```bash
npm run dev
```
If you want to open it using your phone within the same network 
```bash
npm run dev -- --host
```
